## 1) 브랜드 포지션/톤

### 브랜드 포지션
**“경기 지역 고객이 신뢰하고 먼저 문의하게 만드는 판매 브랜드”**

- 단순 상품 노출이 아닌, **문의로 이어지는 상담형 판매 경험**을 제공
- 가격 경쟁 중심이 아니라 **정확한 안내·빠른 응답·안심 거래**를 핵심 가치로 설정
- 랜딩페이지의 1순위 목표는 체류가 아닌 **신규 문의 발생**

### 톤 앤 매너
- **톤**: 실무형, 친절한 신뢰 톤
- **문장 원칙**
- 짧고 명확한 문장
- 추상 표현보다 조건/절차 기반 안내
- 강매형 문구 대신 안내형 CTA
- **지양**
- 과장·단정 표현
- 자극적 긴급 유도 문구
- 감성 위주의 정보 부족 카피

## 2) 핵심 사용자와 시나리오

### 핵심 사용자
1. **비교 탐색형 고객**
- 여러 판매처를 동시에 확인
- 필요: 신뢰 가능한 근거와 빠른 응답 가능성
2. **신중 구매 결정형 고객**
- 가격/배송/교환 등 조건 확인 후 문의
- 필요: 절차의 투명성과 명확한 안내
3. **모바일 즉시 문의형 고객**
- 짧은 체류 후 빠른 액션 선호
- 필요: 직관적 CTA와 최소 입력 폼

### 사용자 시나리오
- **A. 신규 유입**
검색/광고 유입 → 핵심 가치 확인 → 강점/신뢰 확인 → 문의 제출
- **B. 재방문 유입**
재유입 → FAQ/정책/후기 확인 → 불안 해소 → 문의 전환
- **C. 모바일 단기 체류**
상단 CTA 클릭 → 최소 정보 입력 → 문의 완료

## 3) 정보구조(IA)와 페이지 섹션 순서

### IA 설계 원칙
- 흐름: **이해 → 신뢰 → 행동**
- 문의 장벽(정보 부족, 복잡한 폼, 응답 불확실성)을 단계별로 제거
- CTA를 상·중·하단에 반복 배치해 전환 경로 단축

### 페이지 섹션 순서 (Top → Bottom)
1. **헤더**
- 브랜드명, 핵심 한 줄, 문의 CTA
2. **Hero**
- 가치 제안 헤드라인 + 보조 설명 + 1차 CTA
3. **핵심 강점 섹션**
- 빠른 응답/정확한 안내/거래 신뢰/지역 대응
4. **문의 프로세스 섹션**
- 문의 접수 → 조건 확인 → 맞춤 안내
5. **신뢰 증빙 섹션**
- 후기 요약, 운영 기준, 응답 정책
6. **FAQ 섹션**
- 가격·배송·교환/환불·문의 처리시간
7. **최종 CTA 섹션**
- 간편 문의 폼 + 예상 응답 시간
8. **푸터**
- 기본 정보, 연락 채널, 정책 링크

## 4) 시각 시스템(컬러/타이포/레이아웃/모션)

### 컬러 시스템
- **Primary**: Deep Navy/Blue (신뢰·안정)
- **Secondary**: Teal/Blue-Green (친절·지원)
- **Accent**: Amber/Orange (CTA 강조)
- **Neutral**: White/Light Gray (가독성·정돈감)

### 타이포그래피
- **헤드라인**: 굵은 산세리프, 짧은 메시지
- **본문**: 16~18px, 높은 가독성 우선
- **보조 텍스트**: 13~14px, 메타 정보 중심
- **원칙**: 정보 계층(크기/굵기/여백) 명확화

### 레이아웃
- 데스크톱 12컬럼 / 모바일 1컬럼
- 상단은 핵심 가치·CTA 밀도 높게
- 중단은 신뢰 정보 카드화
- 하단은 문의 행동 최적화 블록 구성

### 모션
- 절제된 마이크로 인터랙션 사용
- 섹션 진입 fade-up
- 버튼 hover/focus 반응 강화
- 폼 오류 즉시 피드백
- 과도한 효과(패럴랙스/자동반복 애니메이션) 배제

## 5) 카피 전략(헤드라인 3안, CTA 5안)

### 카피 전략 방향
- “무엇을 판매하는가”보다 “왜 문의해야 하는가”를 선명하게 전달
- 신규 유입 고객의 불안을 줄이는 신뢰형 문장 구조 사용
- 행동 부담을 낮추는 안내형 CTA 우선

### 헤드라인 3안
1. **“비교는 빠르게, 문의는 확실하게.”**
2. **“경기 고객을 위한 정확한 판매 상담.”**
3. **“망설임을 줄이는 안내로 신규 문의를 만듭니다.”**

### CTA 5안
1. **지금 문의하고 빠른 안내 받기**
2. **상품 조건 상담 요청하기**
3. **간편 문의 남기기**
4. **가격·배송 정보 확인하기**
5. **맞춤 제안 받아보기**

## 6) 차별화 포인트(기존 흔한 디자인과 어떻게 다를지)

### 흔한 판매 랜딩의 문제
- 할인/이벤트 중심 구성으로 신뢰 정보 부족
- 템플릿형 비주얼과 유사 카피 반복
- 문의 버튼은 많지만 결정 근거가 약함
- 폼 입력 장벽으로 전환 손실 발생

### 차별화 전략
1. **문의 전환 중심 IA**
- 시각 장식보다 행동 유도 흐름 우선
2. **신뢰형 콘텐츠**
- 응답 기준·절차·정책을 명확히 제시
3. **결과 중심 카피**
- 빠른 안내·정확한 정보·안심 문의를 반복 강화
4. **모바일 우선 UX**
- CTA 재노출 + 최소 입력 폼
5. **절제된 프리미엄 톤**
- 과장 없는 정돈감으로 브랜드 신뢰 확보

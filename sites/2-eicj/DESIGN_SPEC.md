# DESIGN_SPEC.md

## 1) 브랜드 포지션/톤

### 브랜드 포지션
**오픈클로 추가 신청 테스트 2**는 부산 기반 서비스업 브랜드로,
원페이지 안에서 **브랜드 신뢰 형성 → 문의 전환**을 완결하는 구조를 목표로 한다.

- 핵심 가치
- **명확한 소개**: 첫 화면에서 서비스 정체성을 즉시 전달
- **신뢰 증명**: 특징/근거/상세 안내로 불확실성 해소
- **행동 유도**: 자연스러운 CTA 흐름으로 문의 유입 강화

### 톤 앤 매너
- **톤**: 전문적 · 친절한 안내형 · 과장 없는 자신감
- **문장 원칙**
- 짧고 명확한 문장
- 추상어보다 구체 정보(절차/응답시간/범위)
- 압박형 문구보다 제안형 문구
- **지양**
- 보장형/과장형 카피
- 감성 과잉 문구
- 기능 나열형 장문

## 2) 핵심 사용자와 시나리오

### 핵심 사용자
1. **첫 방문 빠른 판단형**
- 니즈: “믿을 수 있는 브랜드인지”를 빠르게 확인
2. **비교 검토형**
- 니즈: 특징/장점 및 프로세스·FAQ 확인 후 문의 결정
3. **모바일 즉시 문의형**
- 니즈: 짧은 동선에서 간편하게 문의 접수

### 사용자 시나리오
- **A. 신규 유입**
- 상단 메뉴 확인 → 대형 히어로에서 가치 인지 → 특징/장점 확인 → 문의 CTA 클릭
- **B. 검토 유입**
- 장점 확인 → 하단 상세설명(프로세스/FAQ/상세안내) 확인 → 문의 제출
- **C. 모바일 유입**
- 메뉴 ‘문의’ 또는 히어로 CTA 이동 → 폼 작성 → 완료

## 3) 정보구조(IA)와 페이지 섹션 순서
*(상단 메뉴 + 대형 히어로 + 특징/장점 + 상세설명 + footer)*

> **원페이지(one-page) 기준 고정 구조**

1. **Header (상단 고정)**
- 메뉴 2~3개: `소개 | 특징 | 문의`
- 우측 보조 CTA 버튼(문의하기)

2. **대형 Hero 섹션**
- 임팩트 헤드라인 + 보조 카피
- 사진/이미지 메인 비주얼 크게 배치 (최소 80~100vh 권장)
- 1차 CTA

3. **특징/장점 섹션 (중단 핵심)**
- 3~4개 카드형 장점
- 각 카드: 장점 + 근거 + 기대효과

4. **신뢰 보강 블록 (권장)**
- 리뷰 요약
- 성과/통계 하이라이트
- 신뢰 배지(응답속도/만족도/진행건수)

5. **하단 상세설명 섹션**
- **프로세스**: 문의 → 상담 → 진행
- **FAQ**: 비용, 기간, 응답 시간, 진행 방식
- **상세안내**: 제공 범위, 준비사항, 유의사항

6. **문의 섹션**
- 간편 문의 폼
- 예상 회신시간/다음 단계 안내
- 최종 CTA

7. **Footer (최하단)**
- 회사 기본 정보, 연락 채널, 정책 링크

## 4) 시각 시스템(컬러/타이포/레이아웃/모션)

### 컬러 시스템
- **Primary**: Deep Navy (신뢰/전문성)
- **Secondary**: Teal 계열 (안정/친절)
- **Accent**: Amber 계열 (CTA/강조)
- **Neutral**: White/Light Gray (가독성 배경)

### 타이포그래피
- **헤드라인**: 굵은 산세리프(임팩트 중심)
- **본문**: 16~18px, 높은 가독성 중심
- **보조 텍스트**: 13~14px, 메타/설명 정보

### 레이아웃
- 원페이지 수직 스크롤 구조
- Hero에 가장 큰 시각 비중 부여
- 특징/장점은 카드 그리드로 빠른 스캔 가능하게 설계
- 상세설명은 스텝/아코디언 방식으로 정보 밀도 조절

### 모션
- Hero 진입: 카피 → 비주얼 → CTA 순차 등장
- 카드 hover/focus 미세 강조
- 섹션 전환 fade-up
- 과도한 패럴랙스/반복 애니메이션 지양

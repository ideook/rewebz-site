## 1) 브랜드 포지션/톤

### 브랜드 포지션
**“대구 기반 서비스 브랜드로서, 신뢰를 먼저 제공하고 행동(문의)으로 연결하는 파트너”**

- 랜딩페이지의 핵심은 ‘소개’가 아니라 **신뢰 누적형 전환 설계**
- 핵심 가치:
1. **명확성**: 서비스 가치가 한 번에 이해됨
2. **신뢰성**: 근거·사례·프로세스로 불확실성 해소
3. **행동성**: 사용자 흐름에 맞춘 CTA로 문의 유도

### 톤 앤 매너
- **톤**: 따뜻한 전문성 + 실무형 명료함
- **카피 원칙**
- 짧고 단정한 문장
- 추상 수식어보다 구체 근거
- “강요”보다 “안내”
- **지양**
- 과장형 보장 문구
- 과도한 감성 과잉
- 의미 없는 트렌디 표현 반복

## 2) 핵심 사용자와 시나리오

### 핵심 사용자
1. **첫 방문 판단형 사용자**
- 니즈: “이 브랜드가 믿을 만한가?”를 빠르게 확인
2. **비교 검토형 사용자**
- 니즈: 타사 대비 차별점과 실행력 검증
3. **모바일 즉시 문의형 사용자**
- 니즈: 긴 탐색 없이 빠른 문의 접근

### 사용자 시나리오
- **A. 신규 유입**
광고/검색 유입 → 대형 Hero 인지 → 특장점/성과 확인 → 문의
- **B. 재방문 유입**
리뷰/FAQ/프로세스 확인 → 의사결정 확신 → CTA 클릭
- **C. 모바일 짧은 체류**
상단 CTA 또는 중간 고정 CTA 클릭 → 간편 폼 제출

## 3) 정보구조(IA)와 페이지 섹션 순서
*(스크롤 볼륨이 충분한 롱페이지 구성)*

1. **Sticky Header**
- 로고, 섹션 앵커, 상시 CTA

2. **대형 Hero (강한 비주얼 중심)**
- 임팩트 카피 + 대형 배경 비주얼 + 1차 CTA
- 신뢰 배지(응답속도/프로젝트 수/만족도)

3. **브랜드 핵심 메시지 하이라이트 블록**
- “우리가 해결하는 문제”를 큰 카드 2~3개로 요약

4. **무드보드/컨셉 이미지 제안 섹션**
- **컨셉 A: Premium Editorial**
- 넓은 여백, 선명한 타이포, 정제된 오브제
- **컨셉 B: Human-Centered Trust**
- 실제 상담/협업 장면, 자연광 톤
- **컨셉 C: Modern Structured**
- 그리드 기반, 블록형 정보 배치, 하이컨트라스트 포인트

5. **특장점(USP) 섹션 — 설득형 구조**
- USP 01: 전략적 기획력
- USP 02: 실행 정확도
- USP 03: 응답/운영 체계
- USP 04: 전환 최적화 방식
- 각 항목: **문제 → 접근 방식 → 결과** 3단 전개

6. **성과/통계 섹션**
- 핵심 숫자(전환율 개선, 평균 응답시간, 누적 프로젝트)
- 큰 수치 + 짧은 설명 + 아이콘/배지 강조

7. **리뷰/레퍼런스 섹션**
- 고객 코멘트 카드 + 업종 태그
- 신뢰 강화를 위한 실명/역할 표기 정책 제안

8. **프로세스 섹션**
- 문의 → 진단 → 제안 → 실행 → 피드백
- 단계별 예상 소요시간 표시

9. **FAQ 섹션**
- 비용 범위, 진행 기간, 수정/커뮤니케이션, 결과 리포트

10. **중간 CTA 리마인드 배너**
- “지금 문의 시 받는 가치”를 단일 메시지로 강조

11. **최종 CTA + 문의 폼 섹션**
- 최소 입력 필드 + 예상 회신 시간 + 다음 단계 안내

12. **푸터**
- 기본 정보, 연락 채널, 정책 링크

### 베리에이션 가이드
- **레이아웃 변주**
- Hero 중앙정렬형 / 좌우 분할형 / 비대칭 이미지형
- **색상 변주**
- Navy+Gold / Charcoal+Sky / Ivory+Emerald
- **타이포 변주**
- Bold Sans 중심 / Serif+Sans 혼합 / Condensed 타이틀형
- **섹션 순서 변주**
- 성과 선배치형 / 리뷰 선배치형 / USP 선배치형

## 4) 시각 시스템(컬러/타이포/레이아웃/모션)

### 컬러 시스템
- **Primary**: Deep Navy (신뢰/전문)
- **Secondary**: Slate Gray (안정/구조감)
- **Accent**: Warm Gold 또는 Bright Teal (강조/CTA)
- **Neutral**: Ivory/Light Gray (가독성)

### 타이포그래피
- **헤드라인**: 존재감 있는 디스플레이 산세리프
- **본문**: 16~18px 가독성 중심
- **강조 텍스트**: 성과/통계는 숫자 시각 강조

### 레이아웃 원칙
- 12컬럼 기반 + 여백 중심 구성
- Hero 최소 90vh 권장
- 스크롤 지점마다 신뢰요소와 CTA 교차 배치

### 모션 원칙
- Hero 진입 시 순차 등장(카피 → 배지 → CTA)
- 카드 hover 시 미세한 depth 변화
- 섹션 reveal은 짧고 절제되게
- 과한 모션/루프 효과 금지

## 6) 차별화 포인트(기존 흔한 디자인과 어떻게 다를지)

1. **롱스크롤 설득 구조**
- 단발성 소개가 아닌 신뢰를 단계적으로 축적
2. **대형 Hero 중심 임팩트**
- 첫 화면에서 감정+논리 동시 설득
3. **무드보드 제안 내장**
- 비주얼 방향성 자체를 전략 자산으로 활용
4. **강조 포인트 체계화**
- 배지/하이라이트/통계/리뷰를 전략적으로 배치
5. **변주 가능한 디자인 시스템**
- 동일 템플릿 반복이 아닌 브랜드별 차별화 가능

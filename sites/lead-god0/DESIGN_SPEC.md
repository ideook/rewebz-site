## 1) 브랜드 포지션/톤

### 브랜드 포지션
**“대전에서 신뢰를 바탕으로 예약까지 연결되는 성장 상담 서비스”**

- 방문자가 정보만 보고 떠나는 것이 아니라, **안심하고 예약까지 진행**하도록 설계된 브랜드
- 핵심은 화려한 표현이 아닌 **전문성의 명확한 전달 + 예약 과정의 부담 최소화**
- “성장”이라는 민감한 주제에 맞춰, 결과 과장보다 **과정의 투명성**을 전면에 배치

### 톤 앤 매너
- **톤**: 신중하고 따뜻한 전문 톤
- **언어 원칙**
- 불안을 자극하지 않고, 걱정을 이해하는 문장
- 짧고 명확한 문장으로 정보 전달
- 근거 없는 단정 표현 배제
- **피해야 할 톤**
- “즉시 효과”, “확실한 개선” 등 확정형 과장
- 공포 마케팅형 카피
- 감성 과잉 문구로 신뢰 저하 유발

## 2) 핵심 사용자와 시나리오

### 핵심 사용자
1. **초기 탐색형 보호자/고객**
- 특성: 검색으로 처음 유입, 정보 부족으로 불안
- 니즈: “여기가 믿을 만한가?”, “예약해도 부담 없나?”

2. **비교 검토형 사용자**
- 특성: 여러 기관을 비교 중, 예약 직전 망설임
- 니즈: 상담 절차의 투명성, 응대 기준, 신뢰 근거

3. **즉시 행동형 모바일 사용자**
- 특성: 짧은 체류 시간, 간단한 프로세스 선호
- 니즈: 빠른 CTA 접근, 최소 입력으로 예약 완료

### 사용자 시나리오
- **시나리오 A (신규 유입)**
- 검색/광고 유입 → Hero 가치 제안 확인 → 강점/신뢰 요소 탐색 → 예약 CTA 클릭
- **시나리오 B (재방문)**
- 후기·FAQ·프로세스 재확인 → 불안 해소 → 예약 폼 제출
- **시나리오 C (모바일 빠른 전환)**
- 고정 CTA 탭 → 필수 항목 입력 → 예약 완료 및 안내 확인

## 3) 정보구조(IA)와 페이지 섹션 순서

### IA 설계 원칙
- 흐름: **인지(무엇을 제공하는가) → 신뢰(왜 믿을 수 있는가) → 행동(어떻게 예약하는가)**
- 예약 이탈 요인(정보 부족, 절차 불명확, 입력 피로) 제거
- CTA는 상·중·하단 다중 배치로 전환 경로 단축

### 페이지 섹션 순서 (Top → Bottom)
1. **헤더**
- 브랜드명, 핵심 신뢰 한 줄, 예약 CTA

2. **Hero**
- 핵심 가치 제안
- 예약 장벽을 낮추는 보조 문구
- 1차 CTA

3. **핵심 강점 섹션**
- 전문성, 상담 체계, 접근성, 응답 속도

4. **상담/예약 프로세스 섹션**
- 예약 신청 → 사전 확인 → 상담 진행

5. **신뢰 증빙 섹션**
- 후기 요약, 운영 원칙, 응대 기준

6. **FAQ 섹션**
- 비용/준비사항/소요시간/예약 변경 등

7. **최종 예약 섹션**
- 간편 예약 폼
- 응답 예상 시간 및 다음 단계 안내

8. **푸터**
- 기본 정보, 연락 수단, 정책/고지 링크

## 4) 시각 시스템(컬러/타이포/레이아웃/모션)

### 컬러 시스템
- **Primary**: 딥 네이비 (신뢰·전문성)
- **Secondary**: 소프트 틸/민트 (안정·안심)
- **Accent**: 웜 앰버/코랄 (예약 CTA 강조)
- **Neutral**: 라이트 그레이/오프화이트 (가독성 기반)

#### 컬러 적용 원칙
- CTA는 Accent 단일 계열로 일관성 유지
- 정보 영역은 저채도 톤으로 안정감 확보
- 강한 강조색은 핵심 행동 요소에만 제한 사용

### 타이포그래피
- **헤드라인**: 굵고 단정한 산세리프, 짧은 문장
- **본문**: 16~18px, 높은 행간으로 가독성 확보
- **캡션/보조**: 13~14px, 과도한 시선 분산 방지
- **원칙**: 정보 계층을 크기·굵기로 구분, 장문 최소화

### 레이아웃
- **그리드**: Desktop 12컬럼 / Mobile 1컬럼
- **구성 전략**
- 상단: 핵심 가치 + 신뢰 포인트 집중
- 중단: 비교·이해를 돕는 설명 구조
- 하단: 예약 행동에 최적화
- **컴포넌트 방향**
- 카드형 반복으로 안정적 정보 인지
- 폼은 필수 항목 최소화

### 모션 시스템
- **기조**: 절제된 신뢰형 모션
- **권장**
- 섹션 등장 시 짧은 fade-up
- 버튼 hover/focus 명확화
- 폼 오류 즉시 피드백
- **지양**
- 과한 줌/패럴랙스/자동 재생 효과
- 정보 탐색을 방해하는 장식 중심 애니메이션

## 5) 카피 전략(헤드라인 3안, CTA 5안)

### 카피 전략 방향
- 예약 전 불안을 줄이는 **안심형 메시지**
- 과장 없이 “지금 예약해도 괜찮다”는 확신 제공
- 프로세스와 응답 기준을 반복 노출해 행동 유도

### 헤드라인 3안
1. **“걱정을 줄이고, 성장 상담은 더 명확하게.”**
2. **“대전에서 안심하고 시작하는 성장 예약.”**
3. **“예약 전 망설임까지 고려한 성장 상담 안내.”**

### CTA 5안
1. **지금 예약 가능 시간 확인하기**
2. **성장 상담 예약하기**
3. **간편 예약 접수하기**
4. **상담 전 궁금한 점 먼저 문의하기**
5. **빠른 안내 받고 예약 진행하기**

## 6) 차별화 포인트(기존 흔한 디자인과 어떻게 다를지)

### 기존 흔한 디자인의 문제
- 감성 이미지 중심, 예약 결정에 필요한 정보 부족
- 템플릿형 구성으로 브랜드 신뢰 차별점 약함
- CTA는 많지만 예약 전 불안 해소 장치 미흡
- 폼 입력 항목 과다로 이탈 증가

### 차별화 전략
1. **예약 의사결정 중심 IA**
- “예쁨”보다 예약 행동까지 이어지는 정보 순서 설계

2. **불안 해소형 콘텐츠**
- 사용자 질문(비용/절차/준비사항)을 선제적으로 해결

3. **절차 투명성 강화**
- 예약 후 무엇이 어떻게 진행되는지 단계별 명시

4. **모바일 전환 최적화**
- 고정/반복 CTA, 최소 입력 폼, 짧은 안내 문장

5. **절제된 프리미엄 비주얼**
- 과장 효과 대신 신뢰감 있는 톤·여백·타이포 중심

## 결론
성장클리닉 테스트 랜딩은 **“신뢰를 먼저 설계하고 예약을 쉽게 만든다”**는 원칙에 집중한다.
대전 지역 사용자의 불안과 망설임을 줄이는 정보 구조, 절제된 시각 시스템, 간결한 예약 UX를 통해 예약 전환율 증가를 실무적으로 달성하는 기획안이다.

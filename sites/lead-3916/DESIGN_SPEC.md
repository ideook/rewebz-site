## 1) 브랜드 포지션/톤

### 브랜드 포지션
**“인천에서 안심하고 예약할 수 있는 신뢰형 치과 상담 브랜드”**

- 환자의 핵심 니즈(통증·비용·시간·안전)에 먼저 답하는 브랜드
- ‘광고형 치과’가 아니라 **설명 잘하는 치과**로 포지셔닝
- 최종 목표: 정보 탐색을 예약 행동으로 전환

### 톤 앤 매너
- **톤**: 차분함, 전문성, 공감
- **카피 원칙**
- 과장보다 사실
- 감성보다 안내
- 불안 조장 금지, 안심 설계 우선
- **지양 표현**
- “무조건”, “완벽”, “즉시 해결” 등 확정형 문구
- 과도한 이벤트/할인 중심 메시지

## 2) 핵심 사용자와 시나리오

### 핵심 사용자
1. **초진 탐색형**
- 검색 유입, 치과 방문 자체에 부담
- 필요: 예약 전 기본 정보와 심리적 안심

2. **비교 검토형**
- 여러 치과 비교 중
- 필요: 차별점, 절차 투명성, 응답 신뢰

3. **재방문 망설임형**
- 이전 방문 후 문의 전 이탈
- 필요: 비용/통증/소요시간 관련 불안 해소

4. **모바일 즉시행동형**
- 짧은 체류, 빠른 예약 선호
- 필요: 상시 CTA + 최소 입력 폼

### 사용자 시나리오
- **A. 신규 유입**
검색 → Hero 핵심 메시지 확인 → 진료/프로세스 확인 → 예약 클릭
- **B. 비교 후 재유입**
후기·FAQ 확인 → 신뢰 확보 → 문의/예약 제출
- **C. 모바일 즉시 전환**
고정 CTA 클릭 → 간편 정보 입력 → 접수 완료

## 3) 정보구조(IA)와 페이지 섹션 순서

### IA 원칙
- 흐름: **불안 완화 → 신뢰 형성 → 행동 촉진**
- 예약 직전 장벽을 단계별로 제거
- 주요 섹션마다 CTA 배치

### 페이지 섹션 순서
1. **헤더**
- 로고/브랜드명, 대표 신뢰 문구, 예약 CTA

2. **Hero**
- 핵심 가치 제안
- 보조 설명(응답/절차/안심 포인트)
- 1차 CTA

3. **핵심 강점**
- 전문 진료, 위생·안전 기준, 상담 방식, 접근성

4. **예약 프로세스**
- 문의 접수 → 사전 안내 → 내원/상담

5. **신뢰 증빙**
- 후기 요약, 운영 원칙, 응답 기준

6. **FAQ**
- 통증, 비용, 소요 시간, 예약 변경 등 핵심 질문

7. **최종 문의/예약 섹션**
- 최소 필드 폼 + 예상 응답시간

8. **푸터**
- 기본 정보, 연락처, 필수 고지/정책

## 4) 시각 시스템(컬러/타이포/레이아웃/모션)

### 컬러
- **Primary**: Deep Navy (신뢰/전문)
- **Secondary**: Mint/Teal (청결/안정)
- **Accent**: Soft Coral/Amber (CTA 강조)
- **Neutral**: White/Light Gray (가독성 기반)

### 타이포
- **헤드라인**: 굵고 간결한 산세리프
- **본문**: 16~18px, 높은 가독성
- **보조 텍스트**: 13~14px, 정보 보조용

### 레이아웃
- Desktop 12컬럼 / Mobile 1컬럼
- 상단 신뢰 정보 집중, 중단 비교 정보, 하단 예약 집중
- 카드형 모듈 반복으로 안정감 확보

### 모션
- 절제된 인터랙션 중심
- 섹션 진입 fade-up
- 버튼 hover/focus 강조
- 폼 에러 즉시 피드백
- 과도한 모션/자동 슬라이드 지양

## 5) 카피 전략(헤드라인 3안, CTA 5안)

### 카피 전략
- ‘치료 효과’보다 ‘예약 안심감’ 중심
- 예약 전 망설임을 줄이는 설명형 카피 우선

### 헤드라인 3안
1. **“걱정은 덜고, 진료 상담은 더 명확하게.”**
2. **“인천에서 안심하고 시작하는 치과 예약.”**
3. **“문의부터 예약까지, 이해하기 쉬운 치과 안내.”**

### CTA 5안
1. **지금 예약 가능 시간 확인하기**
2. **진료 상담 문의하기**
3. **간편 예약 접수하기**
4. **치료 전 궁금한 점 먼저 물어보기**
5. **빠른 안내 받고 예약 진행하기**

## 6) 차별화 포인트(기존 흔한 디자인과 어떻게 다를지)

### 기존 랜딩의 한계
- 이미지/감성 위주, 실제 예약 정보 부족
- 템플릿형 구성으로 브랜드 신뢰 차별점 약함
- 문의 버튼은 많지만 절차 설명 부족
- 폼이 길어 전환 손실 발생

### 차별화 전략
1. **환자 의사결정 중심 IA**
2. **불안 해소 우선 콘텐츠 구조**
3. **예약 절차의 투명한 단계 제시**
4. **모바일 우선 전환 UX(고정 CTA + 최소 입력)**
5. **절제된 의료 신뢰형 비주얼(청결·정돈·안정)**

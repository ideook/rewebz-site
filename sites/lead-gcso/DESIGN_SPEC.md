## 1) 브랜드 포지션/톤

### 브랜드 포지션
- **포지션 정의**: “빠르게 응답하고, 명확하게 안내하며, 안심하고 문의할 수 있는 서울 서비스 브랜드”
- **핵심 가치**
1. **신속성**: 문의 후 빠른 응답
2. **명확성**: 복잡한 설명보다 이해되는 정보 구조
3. **신뢰성**: 과장 없는 증거 중심 커뮤니케이션

### 브랜드 톤 앤 매너
- **톤**: 차분하고 전문적, 과장 없는 자신감
- **화법 원칙**
- 짧고 분명한 문장
- 추상어보다 구체어 사용(예: “최고의 서비스” 대신 “문의 후 24시간 내 답변”)
- 강매형 문구 금지, 안내형 CTA 우선
- **피해야 할 톤**
- 과도한 감성 카피
- 근거 없는 1위/최고 표현
- 클릭 유도형 자극 문구

## 2) 핵심 사용자와 시나리오

### 핵심 사용자
1. **빠른 의사결정형 사용자 (25~45세, 모바일 중심)**
- 특징: 비교 탐색 시간이 짧음
- 니즈: “여기 믿을 만한가?”, “지금 문의해도 되나?”
2. **신중 검토형 사용자 (30~50세, 데스크톱+모바일 병행)**
- 특징: 서비스 범위/절차/비용 감각 확인 후 문의
- 니즈: 리스크 없는 선택 근거
3. **재방문 사용자 (리타게팅/재유입)**
- 특징: 이전 방문에서 문의 직전 이탈
- 니즈: 마지막 불안요소 해소(응답속도, 절차, 신뢰 증빙)

### 사용자 시나리오
- **시나리오 A (첫 방문)**
검색/광고 유입 → Hero에서 가치 인식 → 신뢰 요소 확인 → 하단 폼에서 문의
- **시나리오 B (비교 중 재방문)**
재유입 → 차별화 포인트/후기 확인 → FAQ로 불안 해소 → CTA 클릭
- **시나리오 C (모바일 즉시 문의)**
스크롤 최소화 → 고정 CTA 탭 → 간편 폼 제출(필수 필드 최소화)

## 3) 정보구조(IA)와 페이지 섹션 순서

### IA 원칙
- 단일 랜딩페이지에서 **인지 → 신뢰 → 행동** 흐름을 완결
- 섹션마다 한 가지 목적만 부여
- 모든 주요 구간에 CTA 재노출

### 페이지 섹션 순서 (Top → Bottom)
1. **헤더 (간결 네비 + 1차 CTA)**
2. **Hero (핵심 가치 제안 + 보조 신뢰지표 + 1차 CTA)**
3. **문제-해결 섹션 (고객의 불편 정의 → 해결 방식)**
4. **서비스/프로세스 섹션 (3단계 진행 방식)**
5. **신뢰 증빙 섹션 (후기/수치/응답 SLA/운영 정보)**
6. **FAQ 섹션 (문의 전 이탈 요인 제거)**
7. **최종 CTA 섹션 (간편 문의 폼 + 기대 응답시간 안내)**
8. **푸터 (기본 사업자/연락 정보, 정책 링크)**

## 4) 시각 시스템(컬러/타이포/레이아웃/모션)

### 컬러 시스템
- **Primary**: Deep Navy 계열 (신뢰/전문)
- **Secondary**: Teal/Blue-Green 계열 (안정/친절)
- **Accent**: Warm Amber 계열 (CTA 주목)
- **Neutral**: 저채도 Gray 스케일 (본문 가독성)
- **원칙**
- CTA는 Accent 단일 체계로 통일
- 배경은 고명도 중성, 정보카드는 중명도 대비
- 한 화면 내 강한 강조색은 1~2개만 사용

### 타이포그래피
- **헤드라인**: 굵은 고딕, 짧고 단정한 문장
- **본문**: 가독성 높은 산세리프, 16~18px 범위
- **숫자/지표**: 탭형 숫자 스타일 권장 (가독성 강화)
- **원칙**
- 한 섹션 한 메시지
- 문단 길이 3줄 내외 유지
- 모바일 우선 행간 확보

### 레이아웃
- **그리드**: 12-column 기반, 모바일 1-column
- **구조**: 상단 고밀도(핵심정보), 중단 설명, 하단 행동 유도
- **여백 전략**: 섹션 간 간격을 충분히 주어 “정돈된 신뢰감” 확보
- **컴포넌트 원칙**
- 카드 모듈 반복으로 일관성
- 정보 우선순위에 따라 크기 대비 조절

### 모션
- **기조**: 절제된 마이크로 인터랙션
- **권장**
- 스크롤 진입 시 페이드/슬라이드(짧게)
- 버튼 호버 시 명도/그림자 변화
- 폼 오류 시 즉시 시각 피드백
- **금지**
- 과도한 패럴랙스
- 지속적인 자동 애니메이션
- 콘텐츠 가독성 방해 모션

## 5) 카피 전략(헤드라인 3안, CTA 5안)

### 카피 전략 방향
- “좋아 보임”보다 “믿고 문의할 이유”를 전달
- 헤드라인은 가치+결과를 한 문장에
- CTA는 행동 부담을 낮추는 안내형 문구 중심

### 헤드라인 3안
1. **“문의가 망설여지지 않도록, 신뢰부터 설계합니다.”**
2. **“서울 서비스업 맞춤 랜딩, 전환은 구조에서 시작됩니다.”**
3. **“예쁜 페이지가 아니라, 문의가 남는 페이지를 만듭니다.”**

### CTA 5안
1. **무료 진단 요청하기**
2. **지금 상담 가능 여부 확인하기**
3. **문의 전환 개선안 받아보기**
4. **우리 서비스에 맞는 구성 제안받기**
5. **간편하게 문의 남기기**

## 6) 차별화 포인트(기존 흔한 디자인과 어떻게 다를지)

### 흔한 랜딩의 문제
- 템플릿형 Hero + 추상 카피 반복
- 과장된 그래디언트/효과로 신뢰보다 피로감 유발
- CTA 문구가 공격적이거나 반복만 많음
- 문의 폼이 길어 이탈 증가

### 본 기획의 차별화
1. **신뢰 설계 중심 IA**
- 감성보다 의사결정 정보 우선 배치
2. **증거 기반 카피**
- 응답시간, 프로세스, 범위 등 검증 가능한 문장 사용
3. **불안 제거형 UX**
- FAQ/절차 안내로 문의 직전 이탈 원인 선제 차단
4. **모바일 전환 최적화**
- 스크롤 단계별 CTA 재배치, 폼 최소 입력 설계
5. **절제된 프리미엄 비주얼**
- 과장된 트렌드 효과 대신 정돈된 고신뢰 톤

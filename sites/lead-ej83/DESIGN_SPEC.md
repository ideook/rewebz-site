# DESIGN_SPEC.md

## 1) 브랜드 포지션/톤

### 브랜드 포지션
**리웹즈 와일드카드 전환 점검**은 서울 기반 서비스 브랜드로,
원페이지 안에서 브랜드 신뢰를 형성하고 문의 전환을 유도하는 구조를 목표로 한다.

- 핵심 목표: **preview → 승격 자동화 동작 점검 + 문의 전환 흐름 검증**
- 핵심 가치:
1. **명확한 소개**: 첫 화면에서 서비스 목적 즉시 이해
2. **신뢰성 확보**: 특징/근거/상세 안내로 의사결정 지원
3. **행동 유도**: 자연스러운 CTA 반복으로 문의 유입 강화

### 톤 앤 매너
- **톤**: 전문적, 단정함, 실무형 안내
- **카피 원칙**:
- 짧고 명확한 문장
- 추상어보다 절차/기준/응답 중심 표현
- 압박형보다 제안형 CTA
- **지양**:
- 보장형 과장 문구
- 과도한 감성 카피
- 기능 나열형 장문

## 2) 핵심 사용자와 시나리오

### 핵심 사용자
1. **신규 방문자(일반 리드)**
- 니즈: 브랜드 신뢰 확인 후 빠르게 문의
2. **비교 검토형 방문자**
- 니즈: 특징/장점, 프로세스, FAQ 확인 후 결정
3. **운영 점검 사용자(내부 실무자)**
- 니즈: preview 환경에서 문의 흐름 검증 후 승격 자동화 점검

### 사용자 시나리오
- **A. 리드 유입 시나리오**
- 상단 메뉴 확인 → 대형 히어로 인지 → 특징/장점 확인 → 문의 CTA 클릭
- **B. 신뢰 확인 시나리오**
- 하단 상세설명(프로세스/FAQ/상세안내) 확인 → 최종 문의 제출
- **C. 운영 검증 시나리오**
- preview URL 접속 → 앵커/폼/CTA 동작 확인 → 승격 후 동일 동작 재검증

## 3) 정보구조(IA)와 페이지 섹션 순서
*(상단 메뉴 + 대형 히어로 + 특징/장점 + 상세설명 + footer)*

> **원페이지(one-page) 고정 구조**

1. **Header (상단 고정)**
- 메뉴 2~3개: `소개 | 특징 | 문의`
- 우측 CTA: `지금 문의하기`

2. **대형 Hero 섹션**
- 임팩트 헤드라인 + 보조 카피
- 메인 비주얼(대형 사진/이미지) 중심
- 1차 CTA 버튼

3. **특징/장점 섹션 (중단 핵심)**
- 3~4개 카드형 장점
- 각 카드: 장점 + 근거 + 기대효과

4. **신뢰 보강 블록 (권장)**
- 배지(응답속도/진행건수/고객만족)
- 통계/리뷰/성과 하이라이트

5. **하단 상세설명 섹션**
- **프로세스**: 문의 → 상담 → 진행/결과
- **FAQ**: 비용, 기간, 응답시간, 진행 범위
- **상세안내**: 준비사항, 유의사항, 커뮤니케이션 방식

6. **문의 섹션**
- 간편 문의 폼 + 예상 회신 시간 + 최종 CTA

7. **Footer (최하단)**
- 기본 정보, 연락 채널, 정책 링크

## 4) 시각 시스템(컬러/타이포/레이아웃/모션)

### 컬러 시스템
- **Primary**: Deep Navy (신뢰/전문성)
- **Secondary**: Slate Blue 또는 Teal (안정/기술감)
- **Accent**: Amber (CTA/강조)
- **Neutral**: White/Light Gray (가독성)

### 타이포그래피
- **헤드라인**: 굵은 산세리프, 짧고 강한 문장
- **본문**: 16~18px, 높은 가독성 중심
- **보조 텍스트**: 13~14px, 메타 정보용

### 레이아웃
- 원페이지 수직 스크롤
- 히어로 80~100vh 권장(비주얼 중심)
- 특징 섹션은 카드 그리드로 한눈에 스캔 가능
- 상세설명은 스텝/아코디언 구조로 정보 정리

### 모션
- 히어로 진입: 카피 → 이미지 → CTA 순차 등장
- 카드 hover/focus 미세 강조
- 섹션 전환 fade-up
- 과한 패럴랙스/루프 애니메이션 지양

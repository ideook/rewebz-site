## 2) 핵심 사용자와 시나리오

### 핵심 사용자
1. **빠른 판단형 사용자**
- 첫 인상으로 문의 여부를 판단
- 필요: 핵심 가치와 신뢰 근거 즉시 확인
2. **비교 검토형 사용자**
- 여러 업체 비교 후 문의
- 필요: 차별점, 진행 프로세스, 응답 기준
3. **모바일 즉시행동형 사용자**
- 짧은 체류, 간편 입력 선호
- 필요: 눈에 띄는 CTA, 최소 입력 폼

### 사용자 시나리오
- **A. 신규 유입**
검색/광고 유입 → Hero 확인 → 강점/신뢰 확인 → 문의 제출
- **B. 재방문 검토**
FAQ·프로세스 확인 → 망설임 해소 → CTA 클릭
- **C. 모바일 단기 체류**
상단 CTA 클릭 → 간편 폼 입력 → 문의 완료

## 3) 정보구조(IA)와 페이지 섹션 순서

### IA 설계 원칙
- 흐름: **이해 → 신뢰 → 행동**
- 문의 전환 장벽(정보 부족, 절차 불명확, 입력 피로) 제거
- CTA를 상단·중단·하단 반복 배치

### 페이지 섹션 순서
1. **헤더**
- 브랜드명, 핵심 메시지, 1차 문의 CTA
2. **Hero**
- 가치 제안 헤드라인 + 보조 설명 + CTA
3. **핵심 강점 섹션**
- 차별 포인트 3~4개 카드
4. **문제-해결 섹션**
- 고객의 주요 불편 정의 → 해결 방식
5. **진행 프로세스 섹션**
- 문의 접수 → 상담/확인 → 후속 안내
6. **신뢰 증빙 섹션**
- 후기, 운영 원칙, 응답 기준
7. **FAQ 섹션**
- 비용, 소요시간, 진행 방법, 문의 후 절차
8. **최종 CTA 섹션**
- 간편 문의 폼 + 예상 응답 시간
9. **푸터**
- 기본 정보, 연락 채널, 정책 링크

## 4) 시각 시스템(컬러/타이포/레이아웃/모션)

### 컬러 시스템
- **Primary**: Deep Navy (전문성/신뢰)
- **Secondary**: Teal 계열 (친절/안정)
- **Accent**: Amber/Orange (CTA 주목)
- **Neutral**: White/Light Gray (가독성/정돈감)

### 타이포그래피
- **헤드라인**: 굵은 산세리프, 짧은 핵심 문장
- **본문**: 16~18px, 넉넉한 행간
- **보조 텍스트**: 13~14px, 안내/메타 정보

### 레이아웃
- Desktop 12컬럼 / Mobile 1컬럼
- 상단: 핵심 메시지 집중
- 중단: 신뢰/설명 정보 강화
- 하단: 전환 행동 집중
- 카드형 모듈로 스캔 효율 강화

### 모션
- 절제된 마이크로 인터랙션
- 섹션 진입 fade-up
- 버튼 hover/focus 반응 강화
- 폼 오류/성공 즉시 피드백
- 과한 패럴랙스·자동 반복 애니메이션 지양

## 5) 카피 전략(헤드라인 3안, CTA 5안)

### 카피 전략 방향
- “왜 지금 문의해야 하는지”를 짧고 명확하게 전달
- 신뢰 근거와 행동 유도를 하나의 흐름으로 연결
- CTA는 부담을 낮추는 안내형 문구 사용

### 헤드라인 3안
1. **“문의가 쉬워지면, 전환은 자연스럽게 올라갑니다.”**
2. **“서울 고객이 신뢰하고 바로 선택하는 서비스.”**
3. **“복잡한 설명 없이, 필요한 답을 빠르게 안내합니다.”**

### CTA 5안
1. **무료 상담 문의하기**
2. **지금 문의 가능 여부 확인하기**
3. **우리 상황에 맞는 제안 받기**
4. **간편 문의 남기기**
5. **빠른 안내 요청하기**

## 6) 차별화 포인트(기존 흔한 디자인과 어떻게 다를지)

### 흔한 랜딩의 한계
- 템플릿형 레이아웃 + 추상 카피 반복
- 시각 효과 과다 대비 신뢰 정보 부족
- CTA 수는 많지만 설득 근거 미흡
- 폼 길이 과다로 입력 이탈 증가

### 차별화 전략
1. **전환 중심 IA**
- 미적 연출보다 문의 행동 흐름 우선
2. **근거 중심 카피**
- 응답 기준/절차/지원 범위 명확화
3. **불안 해소 UX**
- FAQ+프로세스로 문의 직전 망설임 제거
4. **모바일 우선 전환 구조**
- 반복 CTA + 최소 입력 폼
5. **절제된 신뢰형 비주얼**
- 정돈된 컬러·타이포·여백 중심

## 1) 브랜드 포지션/톤

### 브랜드 포지션
**“부산 지역 고객이 안심하고 먼저 문의하게 만드는 판매 파트너”**

- 단순 판매가 아닌, **빠른 상담 + 명확한 정보 + 신뢰 가능한 응대**를 제공하는 브랜드로 포지셔닝
- 가격 경쟁보다 **문의 경험의 품질**로 차별화
- 구매 전 의사결정 부담을 줄여, 문의를 자연스럽게 유도

### 톤 앤 매너
- **톤**: 신뢰감, 정돈감, 실무형 친절
- **문장 스타일**:
- 짧고 분명한 문장
- 추상 표현 최소화, 구체 정보 중심
- “강요”보다 “안내” 중심
- **금지 톤**:
- 과장형 1위/최고 표현
- 급박함을 조장하는 자극 문구
- 정보 없는 감성 카피 반복

## 2) 핵심 사용자와 시나리오

### 핵심 사용자
1. **비교 탐색형 고객 (20~40대)**
- 여러 판매처를 비교 후 문의 여부 결정
- 필요: 신뢰 근거, 빠른 응답 가능성

2. **신중 결정형 고객 (30~50대)**
- 조건/절차/사후 응대 확인 후 문의
- 필요: 거래 안정성, 명확한 프로세스

3. **모바일 즉시행동형 고객**
- 광고/SNS/검색 유입 후 짧은 체류
- 필요: 첫 화면 핵심 이해 + 바로 문의 버튼

### 대표 시나리오
- **시나리오 A (신규 유입)**
검색 유입 → Hero 핵심 가치 확인 → 강점/신뢰요소 확인 → 문의 폼 제출

- **시나리오 B (재방문 유입)**
재유입 → FAQ/후기/프로세스 확인 → 불안 해소 → CTA 클릭

- **시나리오 C (모바일 단기 체류)**
첫 화면 CTA 인지 → 최소 입력 폼 작성 → 문의 완료

## 3) 정보구조(IA)와 페이지 섹션 순서

### IA 설계 원칙
- 한 페이지 내에서 **이해 → 신뢰 → 행동** 흐름 완성
- 문의 직전 이탈 포인트(불명확한 정보, 과도한 입력, 불안감) 제거
- 주요 구간마다 CTA 재노출로 전환 경로 단축

### 페이지 섹션 순서
1. **Header**
- 브랜드명, 신뢰 보조 문구, 상단 CTA

2. **Hero**
- 핵심 가치 제안 1문장
- 보조 설명
- 1차 CTA(문의 시작)

3. **핵심 강점 3~4개**
- 빠른 응답, 명확한 안내, 거래 신뢰, 지역 밀착성

4. **문의 프로세스**
- 문의 접수 → 조건 확인 → 맞춤 안내(또는 견적)

5. **신뢰 증빙**
- 후기 요약, 응답 시간 기준, 운영 정보

6. **FAQ**
- 자주 묻는 질문으로 문의 전 불안 해소

7. **최종 CTA + 문의 폼**
- 필수 최소 입력
- 응답 예상 시간 고지

8. **Footer**
- 기본 사업 정보, 연락 채널, 정책 링크

## 4) 시각 시스템(컬러/타이포/레이아웃/모션)

### 컬러 시스템
- **Primary**: Deep Navy (신뢰/전문성)
- **Secondary**: Teal 계열 (친절/안정)
- **Accent**: Orange/Amber 계열 (CTA 집중)
- **Neutral**: Warm Gray 스케일 (가독성 기반)

#### 적용 원칙
- CTA는 Accent 단일 계열로 통일
- 배경은 고명도 중립톤, 정보카드는 대비 확보
- 강조 색상은 페이지 내 1~2개로 제한

### 타이포그래피
- **헤드라인**: 굵은 산세리프, 짧은 문장
- **본문**: 16~18px 가독성 우선
- **보조 텍스트**: 13~14px, 중립 톤
- **원칙**: 과도한 폰트 혼용 금지, 정보 계층을 크기/굵기로 명확히 구분

### 레이아웃
- **그리드**: Desktop 12-column / Mobile 1-column
- **구성**: 상단 정보 밀도 높임, 하단 전환 집중
- **패턴**: 카드형 모듈 반복으로 안정감 유지
- **모바일 우선**: 주요 CTA 고정 또는 반복 노출 권장

### 모션
- **방향**: 절제된 마이크로 인터랙션
- **권장**:
- 섹션 진입 fade-up (짧고 가볍게)
- 버튼 hover/focus 반응 명확화
- 폼 오류 상태 즉시 표시
- **비권장**:
- 과한 패럴랙스
- 자동 재생형 주목 효과 남용

## 5) 카피 전략(헤드라인 3안, CTA 5안)

### 카피 전략
- “지금 문의해도 괜찮다”는 확신을 만드는 문장 설계
- 판매업 특성상 **조건 명확성 + 응답 속도 + 부담 없는 문의**를 핵심 축으로 반복

### 헤드라인 3안
1. **“비교는 짧게, 문의는 확실하게.”**
2. **“부산에서 믿고 문의할 수 있는 판매 상담.”**
3. **“궁금한 건 빠르게, 안내는 정확하게.”**

### CTA 5안
1. **지금 문의하고 빠른 안내 받기**
2. **상품 조건 상담 요청하기**
3. **가격/재고 문의 남기기**
4. **구매 전 확인사항 상담받기**
5. **간편 문의 접수하기**

## 6) 차별화 포인트(기존 흔한 디자인과 어떻게 다를지)

### 기존 흔한 디자인의 문제
- 시각 자극(배너/할인 강조) 중심, 신뢰 정보 부족
- 템플릿형 Hero로 브랜드 고유성 약함
- CTA는 많지만 문의 전 의사결정 정보가 부족
- 폼이 길고 복잡해 전환 저하

### 차별화 전략
1. **전환 흐름 중심 IA**
- 예쁜 화면보다 문의 결정 구조 우선

2. **증거 기반 카피**
- 응답시간/프로세스/운영 기준 등 검증 가능한 정보 제공

3. **부산 지역 맥락 반영**
- 지역 고객 관점의 접근성/응대 이점 강조

4. **문의 장벽 최소화**
- 필수 항목 축소 + 즉시 행동 가능한 CTA 설계

5. **절제된 신뢰형 비주얼**
- 과장 효과 대신 정돈된 프리미엄 인상 제공

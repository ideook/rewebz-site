## 1) 브랜드 포지션/톤

### 브랜드 포지션
**“부산 기반 엔지니어링/제조 파트너로서, 기술 신뢰를 수치와 공정으로 증명하는 브랜드”**

- 브랜드 핵심은 ‘가격 경쟁’이 아니라 **공정 신뢰·품질 일관성·납기 대응력**
- 랜딩 목표는 단순 소개가 아니라 **기술 검토 후 문의까지 이어지는 전환**
- 방문자가 확인해야 할 메시지:
1. 어떤 제조/엔지니어링 역량이 있는가
2. 왜 믿을 수 있는가 (인증, 설비, 품질 데이터)
3. 문의하면 어떤 절차로 진행되는가

### 톤 앤 매너
- **톤**: 기술 중심, 명확함, 책임감 있는 전문 톤
- **문장 원칙**
- 추상 표현보다 수치·규격·프로세스 중심
- 짧고 단정한 문장
- 과장 대신 검증 가능한 정보 제시
- **지양**
- 감성 위주 모호한 표현
- “완벽 보장” 식 단정형 문구
- 근거 없는 업계 1위/최고 표현

## 2) 핵심 사용자와 시나리오

### 핵심 사용자
1. **구매/조달 담당자**
- 니즈: 공급 안정성, 납기 신뢰, 대응 속도
2. **기술 검토 담당(생산기술/설계 엔지니어)**
- 니즈: 사양 적합성, 가공/제작 가능 범위, 품질 기준
3. **의사결정자(팀장/임원)**
- 니즈: 리스크 최소화, 프로젝트 수행 이력, 커뮤니케이션 체계

### 사용자 시나리오
- **A. 신규 리드 유입**
검색/추천 유입 → 대형 Hero에서 전문성 인지 → 설비/품질/성과 확인 → 문의
- **B. 비교 검토 리드**
타 업체와 비교 → 인증/공정/납기 데이터 확인 → FAQ 검토 → 상담 요청
- **C. 모바일 빠른 문의**
현장/이동 중 유입 → 핵심 통계/배지 확인 → 간편 문의 폼 제출

## 3) 정보구조(IA)와 페이지 섹션 순서
*(충분한 스크롤 볼륨의 롱페이지 기준)*

1. **Sticky Header**
- 로고, 주요 섹션 앵커, 상시 CTA(견적/문의)

2. **대형 Hero (풀뷰)**
- 임팩트 카피 + 공장/설비 강한 비주얼 + 1차 CTA
- 신뢰 배지(인증/납기 준수율/프로젝트 수)

3. **컨셉 이미지/무드보드 제안 섹션**
- **무드보드 방향 A: Industrial Precision**
- 메탈 텍스처, 블루프린트 라인, 설비 클로즈업
- **무드보드 방향 B: Clean Tech Factory**
- 밝은 공정 라인, 청결한 작업환경, 품질 검사 장면
- **무드보드 방향 C: Heavy + Reliable**
- 대형 장비, 구조물 디테일, 현장 스케일 강조

4. **핵심 문제-해결 섹션**
- “기존 협력사에서 자주 발생하는 문제” 3가지
- “우리의 해결 방식” 3가지 매핑

5. **특장점(USP) 섹션 — 설득형 구조**
- USP 01: 공정/설비 역량
- USP 02: 품질관리 체계(QC/검수 프로세스)
- USP 03: 납기·일정 대응력
- USP 04: 기술 커뮤니케이션 및 사후 대응
- 각 항목은 **근거(데이터)-방법(프로세스)-결과(효과)** 3단 구성

6. **성과/통계 하이라이트 섹션**
- 납기 준수율, 재의뢰율, 누적 프로젝트, 평균 대응시간
- 큰 숫자 + 짧은 설명 + 아이콘/배지 강조

7. **신뢰요소 섹션**
- 인증/시험 성적/주요 레퍼런스 로고
- 고객 리뷰(업종/담당 직무 포함)

8. **프로젝트 프로세스 섹션**
- 문의 → 기술 검토 → 견적/일정 → 생산/검수 → 납품
- 단계별 소요 시간 및 산출물 표기

9. **FAQ 섹션**
- 최소 발주 수량, 샘플 제작, 리드타임, 품질 보증, 긴급 대응

10. **중간 CTA 배너 (리마인드)**
- “도면/사양 보내면 가능한 범위 바로 안내” 메시지

11. **최종 CTA + 문의 폼**
- 필수 필드 최소화(업체명/담당자/연락처/요청내용/첨부)
- 예상 회신 시간 안내

12. **푸터**
- 회사 정보, 연락처, 운영시간, 정책

### 베리에이션 가이드
- **레이아웃 변주**
- Hero 비주얼 우측형 / 중앙형 / 풀배경형
- **색상 변주**
- Navy-Orange / Charcoal-Cyan / Steel-Blue-Lime Accent
- **타이포 변주**
- Condensed Bold 헤드라인 / 기하학 산세리프 중심
- **섹션 순서 변주**
- 성과 선배치형 vs 특장점 선배치형 vs 레퍼런스 선배치형

## 4) 시각 시스템(컬러/타이포/레이아웃/모션)

### 컬러 시스템
- **Primary**: Deep Navy/Steel Blue (기술 신뢰)
- **Secondary**: Graphite Gray (산업적 무게감)
- **Accent**: Safety Orange 또는 Electric Cyan (강조/CTA)
- **Neutral**: Light Gray/White (가독성)

### 타이포그래피
- **헤드라인**: 강한 인더스트리얼 인상의 볼드 산세리프
- **본문**: 16~18px 가독성 우선
- **데이터 텍스트**: 숫자 강조형 서체 스타일(굵기/폭 차별)

### 레이아웃
- 12컬럼 기반 + 섹션별 리듬감 있는 여백
- 히어로는 90~100vh 권장
- 데이터/리뷰/배지 블록은 반복 가능한 컴포넌트로 구성

### 모션
- Hero 진입 시 텍스트-비주얼 순차 등장
- 통계 블록 hover/focus 미세 강조
- 스크롤 시 섹션 reveal(짧고 절제된 모션)
- 과한 패럴랙스·지속 애니메이션 금지

- 레이아웃/컬러/타이포/순서 변형으로 템플릿 복제감 최소화

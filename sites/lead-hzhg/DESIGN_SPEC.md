## 1) 브랜드 포지션/톤

### 브랜드 포지션
**“한 페이지 안에서 신뢰를 완성하고 문의를 유도하는 서울 기반 서비스 브랜드”**

- 목적: 원페이지 구조로 브랜드 이해부터 문의 행동까지 자연스럽게 연결
- 핵심 가치:
1. **즉시 이해되는 브랜드 소개**
2. **검증 가능한 특징/장점 제시**
3. **간편한 문의 동선과 명확한 다음 단계 안내**

### 톤 앤 매너
- **톤**: 전문적, 친절한 안내형, 과장 없는 신뢰 중심
- **문장 원칙**
- 짧고 명확한 문장
- 추상어보다 구체 정보
- 압박형 CTA보다 제안형 CTA
- **지양**
- 과장/보장형 문구
- 불안 조장 카피
- 장문 기능 나열

## 2) 핵심 사용자와 시나리오

### 핵심 사용자
1. **첫 방문 빠른 판단형**
- 니즈: 상단과 히어로에서 브랜드 신뢰성 즉시 확인
2. **비교 검토형**
- 니즈: 특징/장점과 상세 설명(프로세스·FAQ) 확인 후 문의
3. **모바일 즉시 문의형**
- 니즈: 짧은 동선에서 CTA 클릭 후 빠른 입력

### 사용자 시나리오
- **A. 신규 유입**
상단 메뉴 인지 → 대형 히어로 메시지 확인 → 특징/장점 탐색 → 문의 제출
- **B. 검토 유입**
특징 확인 → 하단 상세설명(프로세스/FAQ) 확인 → CTA 클릭
- **C. 모바일 유입**
메뉴 ‘문의’ 이동 또는 히어로 CTA 클릭 → 폼 제출 → 완료

## 3) 정보구조(IA)와 페이지 섹션 순서
*(원페이지 one-page 구성)*

1. **상단 헤더/메뉴 (2~3개)**
- 메뉴 예시: `소개 | 특징 | 문의`
- 우측 보조 CTA 버튼(문의하기)

2. **대형 히어로 섹션**
- 임팩트 카피(헤드라인 + 서브카피)
- 메인 비주얼(큰 사진/이미지)
- 1차 CTA 버튼

3. **특징/장점 섹션 (중단 핵심)**
- 3~4개 장점 카드
- 각 카드에 “장점 + 근거 + 기대효과” 구조 적용

4. **신뢰 보강 블록 (권장)**
- 리뷰 요약, 숫자 지표, 배지형 신뢰 포인트

5. **하단 상세 설명 섹션**
- **프로세스**: 문의 → 상담 → 진행
- **FAQ**: 비용/기간/응답/진행 방식
- **상세 안내**: 제공 범위, 준비사항, 유의사항

6. **문의 섹션**
- 간편 문의 폼 + 예상 응답시간 + 다음 단계 안내

7. **최하단 Footer**
- 기본 정보, 연락 채널, 정책 링크

## 4) 시각 시스템(컬러/타이포/레이아웃/모션)

### 컬러 시스템
- **Primary**: Deep Navy (신뢰/전문)
- **Secondary**: Teal 계열 (안정/친절)
- **Accent**: Amber 계열 (CTA 강조)
- **Neutral**: White/Light Gray (가독성)

### 타이포그래피
- **헤드라인**: 굵은 산세리프, 짧은 임팩트 문장
- **본문**: 16~18px 가독성 중심
- **보조 텍스트**: 13~14px 메타 정보용

### 레이아웃
- 원페이지 스크롤 구조
- 히어로는 최소 80~100vh로 크게 구성
- 특징 섹션은 카드형으로 시각적 구분 강화
- 상세설명 섹션은 스텝/아코디언 구조로 정리

### 모션
- 히어로 진입 시 카피/비주얼 순차 등장
- 특징 카드 hover/focus 미세 강조
- 섹션 전환 fade-up
- 과도한 모션/지속 애니메이션 지양

## 최종 요약
최신 파이프라인 새테스트 랜딩은 **상단 간결 메뉴 + 대형 히어로 + 중단 특징/장점 + 하단 상세설명 + 최하단 footer**의 원페이지 구조로 설계한다.
브랜드 신뢰를 빠르게 형성하고 문의 전환까지 끊김 없이 이어지는 실전형 원페이지 전략이다.

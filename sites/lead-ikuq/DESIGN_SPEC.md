## 1) 브랜드 포지션/톤

### 브랜드 포지션
**“광주 기반 서비스 브랜드로서, 신뢰를 구조적으로 쌓아 문의로 전환시키는 파트너”**

- 페이지 목적은 단순 소개가 아니라 **신뢰 축적 + 문의 행동 유도**
- 핵심 가치:
1. **명확한 제안**: 첫 스크린에서 서비스 가치 즉시 이해
2. **검증 가능한 신뢰**: 프로세스, 사례, 지표 중심 제시
3. **행동 유도 설계**: 스크롤 흐름에 맞춘 CTA 반복

### 톤 앤 매너
- **톤**: 전문적, 단정함, 따뜻한 안내형
- **문장 원칙**
- 짧고 명확한 문장
- 추상 표현보다 근거 중심 문장
- 압박형 대신 제안형 CTA
- **지양**
- 과장/보장형 카피
- 감성 과잉 문구
- 유행어 중심의 공허한 표현

## 2) 핵심 사용자와 시나리오

### 핵심 사용자
1. **첫 방문 판단형 사용자**
- 니즈: “믿을 수 있는 업체인지”를 빠르게 확인
2. **비교 검토형 사용자**
- 니즈: 타사 대비 차별점/실행력/응답체계 검증
3. **모바일 즉시 행동형 사용자**
- 니즈: 긴 탐색 없이 바로 문의 가능한 구조

### 사용자 시나리오
- **시나리오 A (신규 유입)**
검색/광고 유입 → 대형 Hero에서 핵심 인지 → 특장점/성과 확인 → 문의
- **시나리오 B (재방문)**
리뷰·FAQ·프로세스 확인 → 불안 해소 → CTA 클릭
- **시나리오 C (모바일 단기 체류)**
상단/중간 CTA 클릭 → 간편 폼 제출 → 후속 안내 확인

## 3) 정보구조(IA)와 페이지 섹션 순서
*(스크롤 볼륨이 충분한 롱페이지 구성)*

1. **Sticky Header**
- 로고, 섹션 앵커, 상시 CTA 버튼

2. **대형 Hero (90~100vh)**
- 임팩트 카피 + 강한 비주얼 + 1차 CTA
- 신뢰 배지(응답속도/프로젝트 수/고객만족 등)

3. **핵심 가치 하이라이트 블록**
- “왜 선택해야 하는지”를 3개 카드로 압축

4. **업종 맞춤 컨셉 이미지/무드보드 제안**
- **A안: Trust Editorial**
- 밝은 배경, 여백 중심, 신뢰형 인물/협업 씬
- **B안: Structured Modern**
- 그리드형 구성, 강한 대비, 정보 블록 중심
- **C안: Human + Data**
- 실제 현장/상담 이미지 + 수치 오버레이

5. **특장점(USP) 섹션 — 설득형 구조**
- USP 01: 문제 진단력
- USP 02: 실행 정밀도
- USP 03: 커뮤니케이션 속도
- USP 04: 성과 개선 체계
- 각 항목은 **문제 → 접근 → 결과** 3단 구성

6. **성과/통계 강조 섹션**
- 핵심 지표(전환 개선율, 평균 응답시간, 누적 프로젝트)
- 큰 숫자 + 짧은 설명 + 아이콘/배지 강조

7. **리뷰/레퍼런스 섹션**
- 고객 후기 카드, 업종/직무 태그, 신뢰 라벨

8. **프로세스 섹션**
- 문의 → 진단 → 제안 → 실행 → 리포트
- 단계별 예상 소요시간 표시

9. **FAQ 섹션**
- 비용/기간/수정범위/커뮤니케이션/성과 측정

10. **중간 CTA 리마인드 배너**
- 문의 시 제공 가치(예: 초기 진단, 예상 플랜) 강조

11. **최종 CTA + 문의 폼**
- 최소 입력 필드 + 예상 회신 시간 + 다음 단계 안내

12. **푸터**
- 기본 정보, 연락 채널, 정책 링크

### 베리에이션 가이드
- **레이아웃 변주**: Hero 중앙형 / 좌우 분할형 / 비대칭 카드형
- **색상 변주**: Navy+Mint / Charcoal+Blue / Ivory+Emerald
- **타이포 변주**: Bold Sans / Serif+Sans 혼합 / Condensed 헤드라인
- **섹션 순서 변주**: 성과 선배치형 / 리뷰 선배치형 / USP 선배치형

## 4) 시각 시스템(컬러/타이포/레이아웃/모션)

### 컬러 시스템
- **Primary**: Deep Navy (신뢰/전문)
- **Secondary**: Mint/Teal (안정/친절)
- **Accent**: Amber 또는 Cobalt (강조/CTA)
- **Neutral**: Off-white/Light Gray (가독성)

### 타이포그래피
- **헤드라인**: 존재감 있는 굵은 산세리프
- **본문**: 16~18px 가독성 우선
- **강조 수치**: 숫자/성과 전용 굵기 스타일

### 레이아웃
- 12컬럼 기반 + 섹션 간 리듬 있는 여백
- Hero 중심의 상단 임팩트, 중하단 설득형 블록 반복
- 하이라이트 블록/배지/통계 카드 컴포넌트화

### 모션
- Hero 진입 시 카피→비주얼→CTA 순차 등장
- 카드 hover 시 미세한 depth/brightness 변화
- 스크롤 reveal은 짧고 절제되게
- 과한 패럴랙스/지속 반복 모션 금지

1. **무료 진단 요청하기**
2. **지금 상담 가능 시간 확인하기**
3. **맞춤 제안 받아보기**
4. **간편 문의 남기기**
5. **빠른 안내 시작하기**

## 6) 차별화 포인트(기존 흔한 디자인과 어떻게 다를지)

1. **롱스크롤 설득 구조**
- 단순 소개가 아닌 신뢰를 단계적으로 축적
2. **대형 Hero 중심 임팩트**
- 첫 화면에서 감정+논리 동시 설득
3. **무드보드 제안 내장**
- 결과물뿐 아니라 방향성 자체를 신뢰 자산화
4. **강조 포인트 체계화**
- 배지·하이라이트·통계·리뷰를 전략적으로 배치
5. **변주 가능한 시스템**
- 레이아웃/색/타이포/섹션 순서 변형으로 반복감 최소화

## 최종 요약
분리파이프라인 최종테스트 랜딩은 **대형 히어로 + 롱스크롤 설득 + 신뢰 증빙 블록 강화**를 중심으로,
브랜드 신뢰를 단계적으로 누적해 문의 전환을 높이는 고볼륨 전환형 페이지 전략으로 설계한다.

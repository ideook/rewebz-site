## 1) 브랜드 포지션/톤

### 브랜드 포지션
**“첫 인상에서 신뢰를 만들고, 스크롤 끝에서 문의를 발생시키는 서울 기반 서비스 브랜드”**

- 단순 소개형 페이지가 아니라 **브랜드 신뢰 축적형 전환 페이지**로 설계
- 핵심 가치:
1. **명확한 전문성**
2. **검증 가능한 신뢰 근거**
3. **행동으로 이어지는 CTA 흐름**

### 톤 앤 매너
- **톤**: 프리미엄 + 실무형 + 차분한 확신
- **문장 원칙**
- 짧고 강한 핵심 문장 + 근거형 보조 문장
- 과장보다 수치/프로세스 중심
- 압박형이 아닌 제안형 CTA
- **피해야 할 톤**
- 추상적인 미사여구
- 과도한 감성 호소
- 근거 없는 1위/보장 문구

## 2) 핵심 사용자와 시나리오

### 핵심 사용자
1. **브랜드 신뢰를 먼저 보는 담당자**
- 니즈: “이 팀이 진짜 믿을 만한가?”
2. **비교 검토형 의사결정자**
- 니즈: “다른 업체와 무엇이 다르고, 실제 성과가 있는가?”
3. **즉시 문의형 실무자(모바일 비중 높음)**
- 니즈: “지금 빠르게 문의하고 다음 단계를 알고 싶다”

### 사용자 시나리오
- **A. 신규 유입**
- 광고/검색 유입 → 대형 Hero에서 가치 인지 → 특장점/신뢰요소 확인 → 문의 CTA
- **B. 비교 재방문**
- 리뷰/성과/프로세스 정독 → 리스크 해소 → 최종 CTA
- **C. 모바일 빠른 전환**
- 히어로/중간 고정 CTA 클릭 → 간편 폼 제출 → 후속 안내 확인

## 3) 정보구조(IA)와 페이지 섹션 순서
*(스크롤 볼륨이 충분한 롱페이지 기준)*

1. **Sticky Header**
- 로고, 핵심 내비, 상시 CTA 버튼(문의하기)

2. **대형 Hero (풀뷰/임팩트 비주얼)**
- 강한 헤드라인 + 서브카피 + 1차 CTA
- 배지형 신뢰 라벨(예: 응답속도, 진행 건수, 고객 만족도)

3. **브랜드 무드보드/컨셉 이미지 섹션**
- 업종 맞춤 비주얼 방향 제안:
- **컨셉 A**: Clean Editorial (화이트+네이비, 정제된 오브제/업무 현장 컷)
- **컨셉 B**: Modern Studio (유리/메탈 질감, 구조적 그리드)
- **컨셉 C**: Human Trust (실제 협업/상담 장면 중심)
- 키워드: 신뢰, 구조화, 선명한 전문성

4. **핵심 문제-해결 섹션**
- 고객이 겪는 불편 3가지 → 우리 방식의 해결 구조 3가지

5. **특장점(USP) 섹션 – 설득형 구조**
- USP 01: 전략적 기획력
- USP 02: 실행 정확도
- USP 03: 응대/커뮤니케이션 품질
- USP 04: 전환 중심 개선 사이클
- 각 카드에 “근거/사례/효과” 3행 구성

6. **성과 하이라이트 블록**
- 큰 숫자 통계(예: 문의율 개선, 응답시간, 프로젝트 수)
- 배지/아이콘으로 시각 강조

7. **리뷰/레퍼런스 섹션**
- 짧은 고객 코멘트 + 역할/업종 태그
- 신뢰를 높이는 실명/기업형 표기 규칙 제안

8. **프로세스 섹션 (Step 1~4)**
- 문의 → 진단 → 제안 → 실행/리포트
- 각 단계별 예상 소요시간 표시

9. **FAQ 섹션**
- 비용, 기간, 커뮤니케이션, 산출물 범위, 수정 정책

10. **중간 CTA 리마인드 배너**
- “지금 문의하면 무엇을 받는지” 명확히 제시

11. **최종 CTA + 문의 폼 섹션**
- 최소 입력 필드 + 응답시간 SLA 표기
- 제출 후 다음 단계 안내 문구

12. **푸터**
- 기본 정보, 정책, 연락 채널

### 베리에이션 가이드 (반복 생성 시 변주)
- **레이아웃 변주**: Hero 좌우 분할형 ↔ 중앙 집중형 ↔ 비대칭 카드형
- **색상 변주**: Navy-Teal ↔ Charcoal-Cobalt ↔ Warm Gray-Azure
- **타이포 변주**: 강한 Condensed 헤드라인 ↔ 여백 중심 Serif 혼합
- **섹션 순서 변주**: 무드보드 선배치형 / 문제해결 선배치형 / 성과 선배치형

## 4) 시각 시스템(컬러/타이포/레이아웃/모션)

### 컬러 시스템
- **Primary**: Deep Navy (#0F172A 계열)
- **Secondary**: Trust Teal (#0F766E 계열)
- **Accent**: Electric Amber (#F59E0B 계열)
- **Neutral**: Off-white / Cool Gray

### 타이포그래피
- **헤드라인**: 임팩트 있는 굵은 디스플레이 산세리프
- **본문**: 가독성 높은 산세리프(16~18px)
- **강조 텍스트**: 숫자/성과는 굵기+색상으로 하이라이트

### 레이아웃 원칙
- 12컬럼 기반, 넓은 여백 + 섹션별 강한 위계
- Hero는 최소 85~100vh 볼륨 확보
- 스크롤 지점마다 신뢰요소/CTA 재노출

### 모션 원칙
- Hero 진입 시 텍스트/비주얼 단계적 등장
- 카드 hover 시 미세한 depth 변화
- 숫자/성과 영역은 subtle count-up(개발 가능 시)
- 과한 패럴랙스/산만한 반복 애니메이션 금지

## 6) 차별화 포인트(기존 흔한 디자인과 어떻게 다를지)

1. **롱스크롤 스토리텔링**
- 짧은 소개형이 아니라, 신뢰를 단계적으로 축적하는 구조
2. **대형 Hero 중심 브랜딩**
- 첫 화면에서 감정+이성 모두 설득
3. **무드보드/컨셉 제안 내장**
- “디자인 결과물” 이전에 “디자인 방향성”을 먼저 신뢰시킴
4. **강조 블록 체계화**
- 배지/통계/리뷰/성과를 계획적으로 배치해 설득력 극대화
5. **변주 가능한 디자인 시스템**
- 똑같은 템플릿 반복이 아닌, 브랜드별 레이아웃·색·타이포 변형 가능

## 최종 요약
디자인프롬프트 최종테스트 랜딩은 **대형 히어로 + 롱스크롤 설득 구조 + 신뢰 증빙 강화**를 중심으로,
브랜드 신뢰를 누적해 문의 전환으로 연결하는 고볼륨 전환형 페이지 전략으로 설계한다.
